Introduction

The BitlyBot API allows developers to interact in a programmatic way with the BitlyBot website.

Parameters

message=yourmessage (required)

format=json (By default)

format=xml

callback=callback (Only with json)

REST API

/shorten

Given a message with long url, returns message with shorter one.

Examples

http://bitly-bot.appspot.com/shorten?message=Have a look at http://cnn.com
http://bitly-bot.appspot.com/shorten?message=Google me on http://google.com&format=xml
http://bitly-bot.appspot.com/shorten?message=Post on http://twitter.com&format=json&callback=execute
